Elements SHMUP
==============

> An Arcade/SHMUP game where the player must master all elements to survive!

https://fatboycrew.github.io/elements-shmup/

## Commands

~~~
    git clone git@github.com:FatBoyCrew/elements-shmup.git
    npm install -g gulp
    npm install
    gulp serve
~~~

## Inspiration
